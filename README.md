# COMP3133 Lab 6 - TypeScript and JavaScript Exercises

This repository contains the exercises for Lab 6 of COMP3133, implementing basic classes in JavaScript and TypeScript.

## Exercises

### Exercise 1: Hello World
- `exercise-1/hello-world.js`: Basic JavaScript console output
- `exercise-1/hello-world.ts`: TypeScript version

### Exercise 2: Customer Class
- `exercise-2/customer.js`: Customer class in JavaScript
- `exercise-2/customer.ts`: Customer class in TypeScript with private fields

### Exercise 3: Customer Class (Variation)
- `exercise-3/customer.js`: Another implementation of Customer class in JavaScript
- `exercise-3/customer.ts`: TypeScript version

### Exercise 4: Complete Implementation
- `exercise-4/customer.js` & `customer.ts`: Customer class
- `exercise-4/main.js` & `main.ts`: Main files demonstrating usage

## Running the Code

### JavaScript Files
Run with Node.js:
```bash
node exercise-X/filename.js
```

### TypeScript Files
First install TypeScript globally if not already:
```bash
npm install -g typescript
```

Compile and run:
```bash
tsc exercise-X/filename.ts
node exercise-X/filename.js
```
